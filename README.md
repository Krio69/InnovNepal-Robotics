# 🚀 InnovNepal | Robotics & Engineering E-Commerce

**InnovNepal** is a premium, high-performance frontend solution designed specifically for the robotics and electronics engineering community. This platform provides a seamless interface for students and professionals in **ECE, EE, and Mechanical Engineering** to source high-precision components with ease.

## ✨ Key Features

### 🏗️ Innovative User Interface
* **Industrial Tech Aesthetic:** A sleek, dark-themed UI with "Electric Blue" accents and glassmorphism effects.
* **Professional Hero Section:** High-impact visuals with the slogan: *"Engineering the Future of Nepal: Precision Robotics & Electronics at Your Fingertips."*
* **Smooth Interactivity:** Integrated smooth scrolling, entrance animations, and "Lift-on-Hover" card effects.

### 🛒 E-Commerce Functionality
* **Advanced Search & Discovery:** Functional search bar to quickly find microcontrollers, sensors, or actuators.
* **Smart Sorting:** Organize components by **Price (Low to High / High to Low)** or **Popularity**.
* **Category Filtering:** Quick-access chips for Microcontrollers, Sensors, Power Modules, and Actuators.
* **Deep Product Insights:** Interactive modals for every item including:
    * Detailed Technical Specifications Tables (Voltage, Current, Interface).
    * Local Currency Integration (NPR/Rs.).
    * Placeholders for Engineering Datasheet downloads.

---

## 💻 Tech Stack
* **Architecture:** Component-based Frontend Structure
* **Styling:** Tailwind CSS (Utility-first CSS)
* **Icons:** Lucide-React / FontAwesome
* **Animations:** CSS3 Transitions & Framer Motion logic
* **Layout:** Fully Responsive (Mobile, Tablet, Desktop)
